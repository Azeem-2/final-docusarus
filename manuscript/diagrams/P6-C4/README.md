# P6-C4 Full Humanoid Digital Twin - Diagrams

This directory contains Mermaid diagrams for Chapter P6-C4: Full Humanoid Digital Twin.

## Diagram List

1. **figure-1-full-humanoid.mmd** - 16-DOF full humanoid structure showing lower body (12 DOF), upper body (4 DOF), and optional torso/head.

2. **figure-2-ros2-architecture.mmd** - ROS 2 architecture for digital twin showing nodes, topics, and simulation environment.

## Style Notes

- Diagrams use consistent color palette from global style guide
- Conceptual focus (no detailed technical specifications)
- Emphasis on structure and architecture
- Black & white readable (colors are supplementary)

## Usage

These diagrams support the full humanoid digital twin project in Part 6, showing structure and ROS 2 integration.
