# P5-C7 Case Studies - Diagrams

This directory contains Mermaid diagrams for Chapter P5-C7: Case Studies.

## Diagram List

1. **figure-1-analysis-framework.mmd** - Analysis framework for systematically evaluating humanoid robots across hardware, locomotion, manipulation, HRI, safety, and AI/control.

2. **figure-2-robot-comparison.mmd** - Comparison of three leading humanoid robots: Tesla Optimus, Figure 01, and Boston Dynamics Atlas with their design philosophies and applications.

3. **figure-3-capability-comparison.mmd** - Comparative analysis of capabilities (locomotion, manipulation, HRI, safety) across the three robots.

## Style Notes

- Diagrams use consistent color palette from global style guide
- Conceptual focus (no detailed technical specifications)
- Emphasis on frameworks, comparisons, and analysis
- Black & white readable (colors are supplementary)

## Usage

These diagrams support the case study analysis in Part 5, showing how to systematically analyze and compare real-world humanoid robots.
