# P5-C5 Human–Robot Interaction - Diagrams

This directory contains Mermaid diagrams for Chapter P5-C5: Human–Robot Interaction.

## Diagram List

1. **figure-1-interaction-modalities.mmd** - Interaction modalities: speech, gestures, touch, vision with their characteristics.

2. **figure-2-multimodal-interaction.mmd** - Multi-modal interaction: modality fusion, context awareness, adaptive interfaces.

## Style Notes

- Diagrams use consistent color palette from global style guide
- Conceptual focus (no detailed algorithms)
- Emphasis on interaction modes and integration
- Black & white readable (colors are supplementary)

## Usage

These diagrams support the practical introduction to Human–Robot Interaction in Part 5, showing how humanoid robots communicate and collaborate with humans.
