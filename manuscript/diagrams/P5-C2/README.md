# P5-C2 Bipedal Locomotion - Diagrams

This directory contains Mermaid diagrams for Chapter P5-C2: Bipedal Locomotion.

## Diagram List

1. **figure-1-walking-gait-cycle.mmd** - Walking gait cycle showing stance phase, swing phase, double support, and single support.

2. **figure-2-zmp-control.mmd** - ZMP-based walking control: plan ZMP trajectory → generate CoM trajectory → execute joint trajectories.

3. **figure-3-capture-point-mpc.mmd** - Comparison of walking control approaches: ZMP (reactive), capture point (predictive), MPC (optimized).

4. **figure-4-terrain-adaptation.mmd** - Terrain adaptation strategies: slope walking, obstacle avoidance, uneven terrain.

## Style Notes

- Diagrams use consistent color palette from global style guide
- Conceptual focus (no detailed control algorithms)
- Emphasis on workflows, comparisons, and strategies
- Black & white readable (colors are supplementary)

## Usage

These diagrams support the practical introduction to bipedal locomotion in Part 5, showing how humanoid robots achieve stable, efficient walking.
