# P5-C4 Manipulation & Dexterity - Diagrams

This directory contains Mermaid diagrams for Chapter P5-C4: Manipulation & Dexterity.

## Diagram List

1. **figure-1-grasping-strategies.mmd** - Grasping strategies: power grasp, precision grasp, in-hand manipulation.

2. **figure-2-hand-kinematics.mmd** - Hand kinematics: forward kinematics, inverse kinematics, hand workspace.

3. **figure-3-grasp-planning.mmd** - Grasp planning workflow: object recognition → grasp synthesis → evaluation → execution.

## Style Notes

- Diagrams use consistent color palette from global style guide
- Conceptual focus (no detailed algorithms)
- Emphasis on strategies, workflows, and concepts
- Black & white readable (colors are supplementary)

## Usage

These diagrams support the practical introduction to manipulation and dexterity in Part 5, showing how humanoid robots achieve dexterous manipulation.
